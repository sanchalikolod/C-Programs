# Student Record Management System
A simple student record management system written in C. This program allows users to add student records, display all records, search for a student by roll number and calculate average marks.
## Features
- Add student records
- Display all student records
- Search a student by roll number
- Calculate average marks
- Menu-driven program
## Concepts used
- Structures
- Arrays
- Functions
- Loops
- Conditional Statements
